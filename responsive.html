Responsive Design

Responsive Design er udtrykket for at en hjemmeside bliver bygget til at være responsive. Det betyder at en hjemmeside bliver passet ind til at kunne ses ordenligt på alle desktop størrelser, tablets, og mobiler. Det betyder også at dine funktioner på hjemmesiden er brugervenlig til brugeren.

Hvorfor Responsive?

Hvis man gerne vil have at sin hjemmeside kan række ud til flest mennesker muligt med adgang til internet, så skal man altid overveje at bygge sin side til at være responsive. Det er sådan en forældet ting ikke at have sin side responsive længere. Nogen søgemaskiner vil ikke priotere din side hvis den ikke er mobile venlig. En god note at have når man koder sin hjemmeside, er at altid design i "Mobile First". Det betyder at man designer sin hjemmeside til at først og fremmest blive vidst på en mobile enhed / Tablet. Grundet man gør det, er fordi så mange mennesker i dag bruger deres mobiler og tablets mere til at surfe på nettet, ind folk der bruger deres desktop eller laptop.

Breakpoints og Media Queries

Breakpoints er vores måde at forklare i vores CSS hvornår nogen bestemte classes og elementer skal opføre sig, med en bestemt skærm størrelse. Man skriver hvornår i pixel et "break" skal ske, som betyder at "når siden er så stor her, se sådan her ud". Dette kaldes for media queries. Med media queries kan vi specifikt forklare vores CSS hvornår vores breaks forkommer. Man kan både arbejde i min-width som betyder "Når siden er min. denne størrelse". Og i max-width "Når siden er max. denne størrelse", for at blive specifik med breaks.

CSS Grid

CSS Grid er en ny måde at lave sin hjemmeside på. Med grids kan du forklare i dit CSS hvor dine forskellige elementer er på siden, og på den måde gøre det meget nemmere at arbejde i responsive design.

Grid lader os arbejde i "tabler", så det er derfor lidt genkendeligt for nogen web-udviklere som har været med siden starten. Grid er så smart den den selv "regner" bredder ud, så vi skal ikke tænke på at lege med float osv. Vi starter Grid med at skrive display: grid; i vores css class, som indholder vores content.

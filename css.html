CSS

Hvis HTMl er skeletet af en hjemmeside, så er CSS huden. Med CSS kan vi begynde at blive kreativ. CSS bruges til at ændre synet på vores forskellige ting i HTML. Måske add en smart box med drop shadow, eller ændre font size på vores h1 og p. Måske give en baggrunds farve. CSS er hvor du virkelige kan begynde at blive kreativ med hvad du med en hjemmeside.

Struktur & Opsætning

CSS sættes op på en anderledes måde ind HTML gør. I CSS kan du igennem selectors lave nogen classes eller IDs som du i din HTML giver til bestemte semantiske tags.


Selector: Her fortæller vi vores CSS hvad vi gerne vil ændre i, eller lave en hel ny class eller ID.
Declaration Start: Her starter vi så vores område hvor vi gerne vil skrive vores kode.
Property: I dette område tager vi fat i de elementer vi gerne vil ændre i, eller skabe. Som feks. vores Font størelse, font type, farve osv.
Property/Value Separator: Her afslutter vi vores linje med elementer vi gerne vil arbejde / ændre i.
Value: Her fortæller vi vores element hvad vi ændre i. Som feks. farven "yellow" som vidst på billedet.
Declaration Seperator: Her afslutter vi ændringen i vores property, og kan nu starte en ny og ændre i den hvis det ønskes.
Declaration End: Her afslutter vi flere ændringer i vores selector, og kan der efter starte et nyt / ændre i en ny.

Selectors

Vores selector er hvor vi tager fat i den class, ID, element på vores HTML side vi gerne vil ændre i. Her har kan vi tage fat i et billedet feks. hvor vi gerne vil have at den måske har en bestemt type kant omkring sig, eller måske skal have lidt shadow. Vi kan også bruge Psuedo classes som er beskrevet længere nede.

ID: Et ID er unikt, da et element kun have et ID ad gangen. Og hvis det ID er brugt på et element, så kan de ikke bruges længere på siden. Din side vil ikke blive godkendt af validering hvis du bruger flere af det samme ID på dit dokument.

Class: En class er IKKE unikt, da du kan bruge det flere gange på dine forskellige elementer, og siden selv. Hvis du gerne vil genbruge en bestemt styling til forskellige elementer på din side, skal du bruge en Class.

Andre Selectors: Du kan altid kalde på andre elementer på din side som feks. lt;p> tags, eller lt;ul> lister, i sted for at lave en ny class til det element. W3Schools har en god liste over dem alle.

Pseudo classes

Pseudo classes bruger vi til at forklare hvordan et element under en bestemt situation skal opføre sig. Det kan være at vi gerne vil have at en Div man hover over måske skifter farver når musen er der. Eller måske vil vi gerne have når vi har klikket på et link på vores side, at den viser du har klikket på den før. Du kan også bruge Pseudo til at forklare dit CSS hvilke fonts der kan trækkes fra, og hvor fra de kan trækkes.
Feks:

:hover: - Noget sker når vi hover over vores bestemte element.
:visted: - Noget sker når vi har intergreret med vores bestemte element.

Man kan se hvad man ellers kan bruge dem til inde på W3Schools hjemmeside

CSS Properties

Der findes utrolig mange CSS Properties som du kan bruge til at ændre din selector. Og det er her hvor man kan begynde at blive kreativ med CSS, da hvordan du placere de forskellige values sammen er op til dig. Dog vil en text-allign nok ikke give mening på et billede, men pointen er at hvis man "leger" længe nok med det, så kan du ende med noget rigtig interessant.

W3Schools har en reference liste over dem alle. Ser ingen mening i at skulle selv skrive dem ind her, når nogen allerede har gjordt det. Som min gamle kode lærer fra Medieskolerne altid sagde "Hvorfor vil du gerne opfinde den dybetalerken igen?
